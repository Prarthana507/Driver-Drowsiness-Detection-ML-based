# Driver Drowsiness Detection
This project detects driver drowsiness and yawning in real time using OpenCV and MediaPipe. It calculates Eye Aspect Ratio (EAR) and Mouth Aspect Ratio (MAR) and generates an alert when fatigue is detected.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy
- Pygame

## Features
- Eye closure detection
- Yawning detection
- Real-time webcam monitoring
- Alert sound for drowsiness
